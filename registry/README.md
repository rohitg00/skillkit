# SkillKit Community Registry

This directory contains the community-curated skills registry for SkillKit.

## How to Contribute

1. Fork this repository
2. Add your skill to `SKILLS.md` under the appropriate category
3. Follow the format: `- [skill-name](repo-url) - Brief description`
4. Submit a pull request

## Format

Each entry follows this format:

```markdown
- [skill-name](https://github.com/owner/repo) - Brief description of what the skill does
```

## Categories

Skills are organized into categories. If your skill doesn't fit an existing category, propose a new one in your PR.

## Requirements

- Skill must have a valid `SKILL.md` file
- Description must be clear and concise (under 100 characters)
- Repository must be publicly accessible
- Skill must follow SkillKit conventions
